## English

### CertStudio
**100% Local & Private. Works directly from your folder.**

[![Live Demo](https://img.shields.io/badge/demo-online-brightgreen.svg)](https://joshuachak.github.io/certificate-issuer/)

### 🌟 Key Features
- **Privacy First**: All processing happens entirely in your browser. No student data ever leaves your computer.
- **Batch Processing**: Import recipient lists via Quick Paste (newline separated) or TSV/CSV files.
- **Interactive Editor**: Drag-and-drop placeholders for Names, Dates, and IDs. Full control over fonts and colors.
- **Dual Export Modes**: Choose between Protected (Image PDF) or Interactive (Selectable PDF).
- **ZIP Download**: Automatically packages all individual PDFs into a single ZIP file.

### 🚀 How to Run
1. Download or clone this repository.
2. Double-click the `index.html` file to open it in your browser.
3. No installation or internet required!

### 🔒 Privacy Statement
**Zero Data Transfer**: This application does not have a backend. Your certificates and recipient data are processed locally using JavaScript. We do not track, store, or see your data.
